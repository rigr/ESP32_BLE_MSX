## Pin Assignment

```
Pin 1 ..... D14 = MX0 (Data bus)
Pin 2 ..... D27 = MX1 (Data bus)
Pin 3 ..... D26 = MX2 (Data bus)
Pin 4 ..... D25 = MX3 (Data bus)
Pin 5 ..... 5 Volt
Pin 6 ..... D33 = MX4 (Left Button)
Pin 7 ..... D32 = MX5 (Right Button)
Pin 8 ..... D13 = CS  (Strobe input)
Pin 9 ..... Ground

not connected to sampler:
             D35 = Scan trigger (low for manual scanning)
             D2  = LED (built-in status LED)
             D0  = BOOT button (web interface management)
```
